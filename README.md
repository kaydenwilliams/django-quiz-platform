# QuizQuest — Interactive Quiz & Learning System

## Overview
A full-stack web application built as a collaborative school project. QuizQuest is a 
role-based quiz and learning platform supporting secure authentication, real-time 
scoring, quiz management, and performance tracking.

https://quizquest-interactive-quiz-learning.onrender.com

## Tools
Python, Django, MySQL, HTML, CSS, Bootstrap

## What It Does
- Role-based authentication for admin and standard users with secure session management
- Full CRUD quiz management with categorization and supplemental file uploads
- Real-time scoring and performance tracking per user
- Audit logging for admin and user actions
- Admin dashboards for monitoring quiz activity and user engagement

## Architecture
Three-tier architecture, Django templates for presentation, Django apps for business 
logic, and a relational MySQL database for data persistence.

## Files
- Django backend with ORM models and UI templates
- SQL schema with foreign keys and indexing
- HTML/CSS/Bootstrap frontend
- Admin dashboard views
